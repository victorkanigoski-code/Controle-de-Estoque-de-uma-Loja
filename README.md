# Controle-de-Estoque-de-uma-Pequena
1. Qual é o problema identificado?

O dono da loja pode perder o controle da quantidade de produtos disponíveis, causando falta de produtos ou compras desnecessárias.

2. Onde esse problema acontece?

Em pequenas lojas, mercados, papelarias e depósitos.

3. Quem é afetado por ele?

O proprietário da loja, os funcionários e os clientes.

4. Como um programa de computador poderia ajudar?

O programa poderia:

Cadastrar produtos.
Informar a quantidade em estoque.
Registrar entradas e saídas de produtos.
Mostrar quais produtos estão acabando.
Exibir relatórios do estoque.
